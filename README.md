# 👟 SoleSphere - Premium Sneaker Shop UI

![Flutter](https://img.shields.io/badge/Flutter-%2302569B.svg?style=for-the-badge&logo=Flutter&logoColor=white)
![Dart](https://img.shields.io/badge/dart-%230175C2.svg?style=for-the-badge&logo=dart&logoColor=white)

**SoleSphere** is a high-end, multi-page Sneaker Shop application built with Flutter. It features a professional e-commerce flow, premium minimalist design, and smooth user interactions. 

---

## ✨ Key Features

### 🛒 Complete Shopping Experience
*   **Premium Intro Screen**: Stunning gradient entry point with a "Just Do It" philosophy.
*   **Smart Shop Page**: Browse the latest collections with real-time **Category Filtering** (Running, Basketball, Casual).
*   **Advanced Search**: Instantly find your favorite kicks with the live search bar.
*   **Product Details**: In-depth view for every sneaker, including interactive **Size Selection** and high-quality presentation.
*   **All Sneakers Grid**: A clean, dedicated view to explore the entire inventory.

### 💖 User-Centric Functions
*   **Wishlist/Favorites**: Bookmark your most-wanted sneakers and manage them in a dedicated Favorites tab.
*   **Advanced Cart Logic**: Real-time total price calculation with a professional "Pay Now" checkout flow.
*   **Notification System**: Mock alerts for order tracking, price drops, and account updates.
*   **User Profile**: Comprehensive profile management with access to settings and notifications.

### 🎨 Design & UX
*   **Hero Animations**: Seamless and buttery-smooth transitions between lists and detail views.
*   **Minimalist Aesthetic**: Curated color palette (Grey-Black-White) for a modern, premium feel.
*   **Responsive Layout**: Optimized for various screen sizes.
*   **Interactive Feedback**: SnackBar notifications for cart and favorite actions.

## 🚀 Getting Started

### Prerequisites
- Flutter SDK installed.
- An IDE (VS Code or Android Studio).

### Installation
1.  **Clone the repository**:
    ```bash
    git clone https://github.com/your-username/SoleSphere.git
    ```
2.  **Navigate to project directory**:
    ```bash
    cd SoleSphere
    ```
3.  **Get dependencies**:
    ```bash
    flutter pub get
    ```
4.  **Run the application**:
    ```bash
    flutter run
    ```
